# Epic 21: 3ペインUI & PTY統合 テスト仕様書 (Walkthrough)

本ドキュメントは、Epic 21で実装された「3ペインUI」「PO Agent Sidebar」「Terminal Dock (PTY通信)」の結合テストおよびUI評価のためのチェックシートです。
本フェーズでは以下の項目に従い、機能性とUXの堅牢性を検証します。

## 1. 正常系テスト (Normal Cases)

### 1-1. 3ペインレイアウトとスクロールの独立性
- [ ] **初期表示**: 左(Kanban) 70%、右(AIサイドバー) 30%の比率で画面が分割されていること。
- [ ] **内部スクロール**: 左上のカンバン領域に大量のコンテンツがある場合でも、メインウィンドウ自体はスクロールせず、カンバンペイン内部でのみスクロールバーが表示・動作すること。
- [ ] **ターミナルの最小化/展開**:
  - アプリ起動時、ターミナルはヘッダーのみの「最小化状態」で表示されること。
  - トグルボタン(▲/▼)をクリックすると、下部 40% の高さまで広がり、xtermが表示されること。
  - トグル操作によって左上のカンバン領域が滑らかに追従（縮小・拡大）すること。
- [ ] **サイドバーの開閉**:
  - ヘッダーの「AI Leader」ボタンをクリック、およびサイドバーの「×」ボタンをクリックした際、左カンバンが全画面(100%)まで滑らかに広がること。右ペイン(30%)への切り替えも機能すること。

### 1-2. AIエージェントとのチャット通信
- [ ] **メッセージ送信と応答**: メッセージ送信後、ローディング状態が表示され、バックエンド経由でAIからの返答が正しく描画されること。
- [ ] **オートスクロール**: 返答が生成された際（特に長文の場合）、メッセージリストが自動的に最下部までスクロールされること。
- [ ] **状態の独立性（永続性）**: AIが思考中（ローディング中）にサイドバーを「×」で閉じ、再度開いたとしても、状態が失われずに正常に完了・表示されていること（アンマウントではなく幅0による非表示隠蔽の確認）。

### 1-3. Terminal Dockの実装とPTY通信
- [ ] **マウント時テスト実行**: アプリ起動後もしくはターミナル展開時、`echo Hello PTY` が自動実行され、緑色の文字で正常結果が表示されていること。
- [ ] **リサイズ追従**: 本体のウィンドウサイズや、ペイン幅（サイドバーの開閉等）を変更した直後、xterm の表示列数や行数が崩れず、はみ出しがないよう再計算（`fit()`）されていること。

## 2. 異常系・エッジケーステスト (Abnormal / Edge Cases)

### 2-1. PTY エラーとタイムアウト挙動
- [ ] **無効なコマンド**: 存在しないコマンドや構文エラーを送出した際、適切な標準エラー出力(`stderr`) または エラーメッセージが赤文字等で出力されること（フロントエンドがクラッシュしないこと）。
- [ ] **長時間コマンドのタイムアウト (最大30秒)**: 故意に終わらないコマンド（例として `ping -n 100 127.0.0.1` などを強制発行）を呼んだ場合、Tauri側の30秒タイムアウトで適切にエラーが返され、xtermにタイムアウトエラーとして表示されること。

### 2-2. AI API のエラーとタイムアウト挙動
- [ ] **チャットAPIタイムアウト (最大60秒)**: インターネット切断環境等で故意にチャットを送信した場合、バックエンドの60秒タイムアウトに伴い、フロント側で「推論に失敗しました」等の Toast 表示やUIリセットが正常に行われる。スピナーが無限ループしないこと。

### 2-3. リアクティブ制御のエッジケース（Strict Mode耐性）
- [ ] **ゾンビプロセスの防止**: アプリのリロードやナビゲーションでコンポーネントが一度アンマウントされたのち再マウントされた際、過去のPTYの残骸（無駄な表示処理など）が重複せず、クリーンに一回だけセッションが開始されること。

## 3. 次ステップでのUI/UX磨き込み要件 (Phase 4 後半向け)
このテスト仕様にすべて合格したことを確認後、以下のUI装飾を追加して最終完了とする。
- Tailwind v4を用いたGlassmorphism（背景のぼかし、半透明）、影（Shadows）の調整。
- マイクロアニメーション（Hover時の色変化、トグル開閉時のイージング等、Premium な手触り）の追加。
