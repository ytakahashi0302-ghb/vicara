# EPIC49: POふせん機能（POサイドバー統合）

## 背景

DEV実行中にPOが気づいたことや改善点をMarkdown形式でメモできるドキュメント機能が必要である。これらのふせんはレトロスペクティブでの振り返りの入力源となる。POアシスタントサイドバー内に「チャット / ふせん」タブ切替を追加し、会話と近い位置でメモを管理できるようにする。

## ゴール

- POアシスタントサイドバーにふせんタブを追加する
- 軽量にすぐ書けるふせん管理UIを実装する
- ふせんからレトロアイテムへの転記機能を実装する

## スコープ

### 含む

- `src/components/ai/PoAssistantSidebar.tsx` へのタブ切替追加
- `src/components/ai/NotesPanel.tsx` の新規作成
- ふせんの一覧表示・作成・編集・削除UI
- Markdown記法を含む本文表示
- ふせん → レトロアイテム転記ボタン

### 含まない

- POアシスタントAIからの自動ノート追加（EPIC52で実装）
- SM AI合成でのノート参照（EPIC51で実装）
