- **TaskChatPane（旧: タスク個別チャット）** → 完全削除

## 1. UI動作確認手順

### Step 1: サイドパネルの開閉（トグル）

1. アプリを起動します（`npm run tauri dev`）。
2. ヘッダー右側に **「AI Leader」ボタン**（Botアイコン付き、インディゴ色の背景）が表示されていることを確認します。
3. **初期状態は「閉じた状態」** です。メインコンテンツ（バックログ/ボード）が画面幅いっぱいに使われていることを確認します。
4. 「AI Leader」ボタンをクリックします。
5. **右側にサイドパネルがスライドイン**し、幅 `380px` で表示されることを確認します。
6. ボタンの見た目が **アクティブ状態**（より濃いインディゴ背景 + シャドウ）に変わることを確認します。

### Step 2: レイアウトの押し引き

1. サイドパネルが開いた状態で、メインの**カンバンボード / バックログ領域が左側に押し込まれ、サイドパネルの下敷きになっていないこと**を確認します。
2. 「プロダクトバックログ」タブと「アクティブスプリント」タブを切り替え、**どちらの画面でもサイドパネルが干渉せずに正常に表示される**ことを確認します。
3. サイドパネル右上の ✕（閉じる）ボタンをクリックします。
4. サイドパネルが閉じ、メインコンテンツが再び**画面幅いっぱいに戻る**ことを確認します。
5. ヘッダーの「AI Leader」ボタンの見た目が**非アクティブ状態**に戻ることを確認します。

### Step 3: サイドパネルのUI詳細

1. サイドパネルを再度開き、**ヘッダー部**に以下が表示されていることを確認します：
   - ✨アイコン（Sparkles）
   - タイトル: **「AI Team Leader」**
   - サブタイトル: **「Scrum Master & Lead Engineer」**
   - ✕（閉じる）ボタン
2. チャット履歴が空の状態で、**空状態のプレースホルダー（Botアイコン + 説明テキスト）**が中央に表示されていることを確認します。
3. **入力エリア**が下部に固定されていることを確認します。プレースホルダーに「メッセージを入力... (Ctrl+Enter で送信)」と表示されていることを確認します。
4. 入力欄にテキストを入力し、右下の**送信ボタン（紙飛行機アイコン）**がアクティブになることを確認します。
5. `Ctrl+Enter`（Mac: `Cmd+Enter`）で送信できることを確認します。

### Step 4: タスク編集モーダルのロールバック確認

1. スプリントボード画面で任意のタスクカードをクリックします。
2. **TaskFormModal** が開き、以下の状態であることを確認します：
   - シンプルな**1ペイン**レイアウト（右ペインのチャットエリアは存在しない）
   - 「担当者 (Assignee)」ドロップダウンが**存在しない**
   - タイトル、詳細（Markdown対応）、ステータスの3フィールドのみ

## 2. 実装ハイライト

### レイアウトアーキテクチャ
```
App.tsx
├── header (sticky, z-20)
│   ├── Logo / ProjectSelector / ProjectSettings
│   ├── Inception Deck ボタン
│   ├── 履歴ボタン
│   └── 🤖 AI Leader トグルボタン ← NEW (isSidebarOpen を制御)
│
├── div.flex-1.flex (メインコンテンツ + サイドバー)
│   ├── main.flex-1 (ScrumDashboard)
│   └── TeamLeaderSidebar (w-[380px] or w-0) ← NEW
│
└── HistoryModal / Toaster / DevTools
```

### 状態管理
- `isSidebarOpen` 状態は `AppContent` コンポーネントでリフトアップ管理。
- ヘッダーのトグルボタンから `setIsSidebarOpen` を呼び出し、サイドバーコンポーネントへ `isOpen` と `onClose` を prop で渡す設計。

### デザイン
- **カラーテーマ**: インディゴ基調（`indigo-50` 〜 `indigo-700`）で統一。
- **アニメーション**: `transition-all duration-300 ease-in-out` でスムーズな開閉。
- **ユーザーメッセージ**: `bg-indigo-600 text-white` の吹き出しスタイル。
- **AIメッセージ**: `bg-white border shadow-sm` + Markdown完全対応（prose クラスでスタイリング）。
